# sample-react-router-generouted

sample that automatically generates `router.ts` using `src/pages/*`.

## Getting started

### Install

```sh
pnpm install
```

### Usage dev

```sh
pnpm dev
```

### Usage lint

```sh
pnpm lint
```

### Usage build

```sh
pnpm build
```

### Usage preview

```sh
pnpm apps:front preview
```
